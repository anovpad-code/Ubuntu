Project ini berisi dokumentasi penggunaan *Ubuntu 24.04 LTS* di *WSL2 (Windows)*  
sebagai environment belajar *Linux & Python*.

##  Platform
- Windows 10
- WSL2

##  Sistem
- OS: Ubuntu 24.04.3 LTS
- Kernel: WSL2
- Shell: Bash
- Terminal: Windows Terminal

##  Tools
- WSL
- Ubuntu
- Python 3.12
- Nano
- Neofetch

##  Fitur
- Linux berjalan di Windows
- Cocok untuk belajar Python
- Ringan & stabil
- Tidak perlu dual boot
